Network plan

  • Pi runs hotspot (hostapd) with WPA2-PSK.
  • DHCP on 10.10.0.0/24 via dnsmasq. Pi at 10.10.0.1.
  • Static site on :80, MJPEG stream on :8080.
  • Heatmap JSON at /var/www/html/data/motion-today.json.

Security notes

  • LAN-only by default. Do not port-forward.
  • Share a strong passphrase only with trusted roommates.
  • Keep the Pi updated; treat the stream as sensitive.
